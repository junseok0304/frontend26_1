## bash 명령어
1. 디렉토리 생성 : mkdir 디렉토리명
2. 하위 디렉토리로 이동 : cd 디렉토리명
3. 현재 디렉토리의 파일 목록 출력 : ls
4. 상위 디렉토리로 이동하기 : cd ..
5. 디렉토리 삭제하기 : rm -rf 디렉토리명 // 디렉토리와 그 아래 파일들이 모두 삭제된다. 
6. 파일 삭제하기 : rm 파일명 
7. 파일 복사하기 #1 : cp 소스_파일 새_파일
8. 파일 복사하기 #2 : cp 소스_파일 대상 폴더 
9. 디렉토리 복사하기 : cp -r 소스_디렉토리 대상_디렉토리 

10. 현재 디렉토리의 파일 목록 출력 : ls -al
11. 현재 디렉토리 아래의 chap01 디렉토리로 들어가는 명령 : cd chap01

12. js 파일을 쉘에서 실행하기 위한 명령 : node 파일명 
13. node hello1.js (예시임)

# package.json 파일의 내용 
- 프로젝트 이름, 버전, 설명, 실행시 시작될 첫 파일, 빌드 및 테스트를 실행하기 위한 명령, 검색키워드, 작성자, 라이센스 정보, 필요한 라이브러리들 목록
- node_modules 폴더 : 필요한 라이브러리들이 자동으로 저장되는 폴더 
- .eslintc.json 파일 : eslint 설정 파일
- package-lock.json : 프로젝트에 필요한 라이브러리들의 정확한 버전이 자동으로 등록되는 파일, 자동으로 생성된다.
